This table shows how each user responded to each question in each of their attempts at the quiz.

The responses are color-coded as follows:

  • Correct answers
  • Ignored answers (if any)
  • Wrong answers (if any)
  • score% (hints,clues,checks)
    • score is the percentage score for this question
    • hints is the number hints requested
    • clues indicates whether the clue was shown (=1) or not (=0)
    • checks is the number checks performed on this question. A value of "-1" means that the student clicked the "Show answer" button
