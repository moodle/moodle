Envoyer aux enseignants les alertes par courriel

Si cette option est activée, les enseignants reçoivent un bref message lorsque les étudiants ajoutent ou mettent à jour un document.

Seuls les enseignants ayant l'autorisation de noter le travail reçoivent le message. Ainsi, par exemple, si le cours utilise des groupes séparés, les enseignants restreints à un groupe particulier ne reçoivent pas les alertes concernant les étudiants des autres groupes.

Pour les activités hors ligne, aucun message n'est jamais envoyé, puique les étudiants ne remettent aucun document.
