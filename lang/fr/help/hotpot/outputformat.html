Format d'affichage

Vous pouvez choisir entre différents formats d'affichage de votre test

Meilleur
Le test sera affiché dans le meilleur format possible pour le navigateur de l'utilisateur.
v6+
Le test sera affiché en format glisser-déposer pour les navigateur v6+
v6
Le test sera affiché en format pour les navigateurs v6
v5
Le test sera affiché en format pour les navigateurs v5
v4
Le test sera affiché en format pour les navigateurs v4
v3
Le test sera affiché en format pour les navigateurs v3
Flash
Le test sera affiché à l'aide de Flash
mobile
Le test sera affiché dans un format adéquat pour la visualisation d'appareils mobils tels que les téléphones cellulaires.
