Page web en code HTML

    Ce type de ressource permet de développer simplement une page web à l'intérieur de Moodle, tout spécialement lorsque vous utilisez l'éditeur WYSIWYG intégré. Vous pouvez ainsi utiliser tous vos talents d'éditeur HTML, y compris le Javascript.

    La page est enregistrée dans la base de données, et non comme un fichier.
