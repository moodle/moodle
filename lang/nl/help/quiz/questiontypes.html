Een nieuwe vraag maken

Je kunt verschillende soorten vragen toevoegen aan een categorie:

Meerkeuzevragen

    Als antwoord op de vraag (waarbij een afbeelding kan getoond worden) krijg je meerdere mogelijkheden aangeboden. Er zijn twee types meerkeuzevragen vragen: slechts één antwoord of meerdere antwoorden toegestaan.

    Meer informatie:

Kort antwoord

    Als antwoord op de vraag (waarbij een afbeelding kan getoond worden) moet je een woord of een zin typen. Er kunnen meerdere mogelijke antwoorden gegeven worden, eventueel met verschillend puntenaantal. Antwoorden kunnen al dan niet hoofdlettergevoelig zijn.

Waar/Onwaar

    Als antwoord op de vraag (waarbij een afbeelding kan getoond worden) kun je uit twee mogelijkheden kiezen: waar of onwaar

Willekeurige set

    Geeft je de mogelijkheid om een aantal vragen uit een grote set aan te bieden. Bij elke poging worden de vragen willekeurig gekozen, zodat de leerling telkens een andere test krijgt aangeboden.

    Meer informatie:

Willekeurig korte antwoorden paren

    Geeft je de mogelijkheid om een aantal willekeurige korte antwoordvragen te presenteren als koppelvragen.
