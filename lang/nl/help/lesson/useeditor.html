Gebruik van de HTML Editor

Door deze optie in te schakelen kun je de HTML editor (vanaf hier editor genoemd) gebruiken in de plaats van gewone tekstvelden. Met de editor kun je gemakkelijk tekst opmaken en gemakkelijk iets meer "complexe" opmaak toevoegen zoals hyperlinks.

De editor kan in- en uitgeschakeld worden door te klikken in het vakje naast Gebruik editor. De editor hangt samen met elk item afzonderlijk. De editor inschakelen voor, bijvoorbeeld, Respons 1 schakelt de editor alleen in voor dit item en schakelt hem niet in voor alle andere responses.

Als de editor nodig is voor sommige delen van een pagina, klik dan eerst op de Gebruik editor knop (zet een vinkje in het vakje). De pagina moet dan opnieuw worden opgehaald om de editor te laden. Dit kan gedaan worden door op de Toon pagina opnieuw knop onderaan de pagina te drukken.

Merk op dat je toch voorzichtig moet zijn bij het gebruik van de editor. Meestal zullen er zich geen problemen voordoen, maar het is niet aan te raden de editor te gebruiken voor antwoorden van kort antwoordvragen en numerieke vragen. De editor kan extra "onzichtbare" HTML code toevoegen die het verbeteren van de antwoorden in de war kan sturen. De beschrijvingen in vertakkingstabellen worden gebruikt in knoppen en ook daar kan HTML code problemen veroorzaken. Er zijn geen problemen te verwachten met HTML in antwoorden die alleen gebruikt worden om te tonen, zoals in meerkeuzevragen. Ook in de respons zou het gebruik van de editor geen problemen mogen opleveren.

Als je vermoedt dat er een probleem is met een tekstonderdeel, dan kun je gewoon de editor afzetten door het vinkje uit het Gebruik editor vakje weg te klikken. Wanneer de pagina opnieuw geladen wordt, dan zie je de "ruwe" tekst in een gewoon tekstveld. Je kunt dan de tekst bewerken. De editor meerdere keren in- en uitschakelen voor een bepaald stuk tekst zou de tekst zelf niet mogen wijzigen.

De "status" van de Gebruik editor optie wordt bij elk onderdeel bewaard. Wanneer je pagina waarvoor je de editor gebruikt hebt opnieuw bewerkt, dan zal deze pagina opnieuw in de editor verschijnen.

Tenslotte nog even opmerken dat het gebruik van de editor nogal "duur" is, uitgedrukt in gebruikte bronnen binnen je browser. Als je voor bepaalde onderdelen de editor niet nodig hebt, dan zul je sneller en met minder bronproblemen kunnen werken door de standaard tekstvelden te gebruiken.
