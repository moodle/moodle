Wenn Sie als "Dateinamen" eine Datei angeben, Wird die Datei als Beginn einer eine von Tests interpretiert. Alle Tests der Reihe werden zum Kurs mit identischen Einstellungen hinzugefügt.

Wennn der "Name" ein Ordner ist, werden alle Tests des Ordners als Reihe von Tests zum Kurs mit identischen Einstellungen hinzugefügt.
