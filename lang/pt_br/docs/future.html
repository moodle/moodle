Futuro

Com o Moodle ganhando maturidade, seus caminhos são influenciados fortemente pela comunidade de colaboradores e usuários. Uma base de dados dinâmica de características propostas e seu status podem ser encontrados em moodle.org/bugs. Suas contribuições em forma das idéias, de código, de avaliação e de promoção são todas muito bem vindas - veja o Manual para Desenvolvedores e os fóruns da comunidade para mais detalhes. Você pode também, pagar para ter determinadas características desenvolvidas rapidamente - veja em moodle.com/development para informações.

Segue o plano atual para o futuro, tenha sempre em mente que este planejamento depende de doadores e desenvolvedores.

Versão 1.6

Blogs como principal ferramenta para atividades reflexivas, com alimentadore RSS a nível de usuário, curso e site.

Módulo Base de dados, ferramenta de uso geral para entra de dados colaborativa, procura e navegação.

Integração com LAMS (Leaning Activity Management System) http://www.lamsfoundation.org/ como uma atividade ou formato curso.

Melhores estatísticas.

Gestão de entrada de mensagens de correio-eletrônico.

Serviços Web API básicos.

Versão 1.7

Página "Meu Moodle" provendo uma visão customizável de todos os cursos, acrescidos de leitores de RSS feeds.

Página Perfil do Usuá melhorada, integrando Blogs, Alimentadores, etc em uma página web semi-púbica.

Suporte preliminar para IMS LD nível A, permitindo importação e exportação.

Suporte completo ao SCORM 2004.

Serviços Web API melhorados.

Integração com alguns repositórios.

 

Versão 2.0

Atividades condicionais, permitindo condições e caminhos forçados.

Suporte completo para IMS LD padrão.

Grupos aprimorados, permitindo que grupos sejam definidos no site e a nível de atividade.

Implementação melhorada de perfis de usuários, permitindo perfis customizados a nível de site, curso e atividade.

Muitas das páginas principais customizáveis usando padrões HTML (templates).

Implementações orientadas a viabilizar o Moodle como sistema de colaboração distribuida em rede como uma evolução natural do foco de Moodle na colaboração. Aguardem as novidades.
